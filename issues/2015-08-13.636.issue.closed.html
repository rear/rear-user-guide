#636 Issue closed: Fedora23 ISOLINUX does not boot¶

Labels: bug

gdha opened issue at 2015-08-13 10:45:¶

The CD image on Fedora23 (alpha) contains:

# ls /mnt/isolinux/
boot.cat  initrd.cgz  isolinux.bin  isolinux.cfg  kernel  message  rear.help

but the syslinux version is 6.03 which means we need also ldlinux.c32 and on fedora23 it is present under directory /usr/share/syslinux (not yet modular as under Ubuntu 15 and Debian 8). rear thinks it is legacy syslinux and do not copy the ldlinux.c32 to the isolinux CD boot image.
