gdha commented at 2017-01-23 08:53:¶

@Veklip As /mnt mount point used in the /etc/rear/local.conf file is part of the / file system (and VG /dev/mapper/vg_sys or /dev/sda3) you cannot mount and reformat the same partition. Please use and external disk (e.g. USB devices or NFS mount point) otherwise you will never be able to re-format /dev/sda3.
If on /mnt file system you mounted another NFS file system, then you forgot to tell me.
