I would like to see rear gain support for backing up and recovering ZFS disks including root on ZFS disks eg Proxmox VE and Ubuntu are two distros that let the user easily install to a ZFS root pool. I just grepped the rear source tree for zpool and zfs and got no results so I think its a safe assumption to make that it currently has no support for ZFS.

I would recommend using syncoid (part of sanoid) for replicating ZFS snapshots.
