jsmeix commented at 2020-08-04 11:36:¶

As far as I know migrating things like more than one IP per interface
were never supported by ReaR so this issue is an enhancement request.

We have already
https://github.com/rear/rear/issues/2312
which had emerged from
https://github.com/rear/rear/issues/2310
and
https://github.com/rear/rear/pull/2313

According to
https://github.com/rear/rear/issues/2312
such enhancement requests will not be implemented by ReaR upstream
on a voluntary base "just for free" but need substantial sponsorship, cf.
http://relax-and-recover.org/development/
