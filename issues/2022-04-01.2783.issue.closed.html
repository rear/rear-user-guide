pcahyna commented at 2022-04-01 09:32:¶

Sorry, it is a bit unfortunate, ReaR needs the grub2-efi-x64-modules package, but only on EFI machines, therefore the package does not bring it in automatically.
A trick, if you know what files are missing (the messages mention that ReaR needs /usr/*/grub*/x86_64-efi/moddep.lst and /usr/lib/grub/x86_64-efi/modinfo.sh), you can determine which package you need to install using yum repoquery:

yum repoquery --whatprovides  /usr/lib/grub/x86_64-efi/modinfo.sh

or

yum repoquery --whatprovides  '/usr/*/grub*/x86_64-efi/moddep.lst'

Both commands should print grub2-efi-x64-modules-1:2.02-122.el8.noarch (or something similar).
